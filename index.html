Programação na Web

Na Escola Comercio de Lisboa, na disciplina de Tecnologia 1 no 3ºBTIS, iremos aprender Programação Web 😅 e abordaremos os principios básicos associados para o desenvolvimento para a web, apenas do lado front-end, lado visual. Também sabemos que existe um lado back-end, que é o lado do servidor, mas não iremos abordar esse lado.

Aprender-se-ão as seguintes tecnologias 🋔:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar os conhecimentos, serão feitos alguns laboratórios.
